## 맛집 몬스터 백엔드
[프론트엔드](https://github.com/techeer-f5/jmt-monster-frontend) | 백엔드

친구끼리 공유하는 수집형 맛집 어플리케이션 "맛집 몬스터"!

사용자끼리 친구를 만들고, 친구 지도로 놀러가서 리뷰를 남기는 프라이빗 맛집 공유 어플리케이션입니다.

- Gradle 7.4.1

- Querydsl 5.0.0

- Lombok
- Jackson
- JUnit 5, Mockito, Hamcrest
